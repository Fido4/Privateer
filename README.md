# Privateer

Privateer is a simple Windows screenshot tool in the spirit of Greenshot, but with a slimmer feature set and a more modern UI.

It focuses on the core workflow:
- capture a region
- save or copy it quickly
- open it in a built-in editor for lightweight markup and obfuscation

The app is built with C# and WPF and is intended to feel fast, clean, and easy to use without the extra complexity of a larger screenshot suite.
